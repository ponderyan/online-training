四、关键功能需求（已确认）
================================================================

【题库管理】
- 列表字段：试题内容 / 题型 / 难度 / 科目 / 入库时间 / 来源 / 引用次数
- 筛选：科目 / 题型 / 难度 / 来源
- 分页：页码 + 每页条数 + 跳页
- 批量导入：Excel模板 + 下载模板入口

【试题录入（分题型动态表单）】
- 单选题：题干 + 选项A/B/C/D + 正确答案（单选）
- 多选题：题干 + 选项A/B/C/D/E + 正确答案（多选勾选）
- 判断题：题干 + ✓/✗
- 填空题：题干用 {{_}} 标记填空 → 自动解析 → 逐空输入答案
- 简答题：题干 + 参考答案 + 得分要点
- 案例题：案例背景 + N个子问题（每个子问题含问题+答案）

【智能组卷】（核心逻辑）
- 先选科目 → 自动生成试卷编号（DT+科目-日期-序号），用户可改
- 自由输入总分值（默认100）、考试时间（默认90）
- 用户勾选本次考试要用的题型（未勾选的不显示）
- 每种题型：题数 × 每题分值 = 小计，全部用户自定义
- 总分联动校验：题型小计 ≠ 总分值时提示错误，阻止生成
- 难度比例：易(绿) / 较易(金) / 较难(棕) / 难(红)，点击调整
- 章节分布：均匀分布 / 按权重分布 / 随机分布
- 试题来源混合比例：科目专用% + 公共% 滑块调节
- 生成后预览 + "换一题"功能（同难度同章节替换）

【试卷管理】
- 列表字段：名称 / 编号 / 命题人 / 生成时间 / 修改时间 / 总分题数 / 是否正式考卷
- 状态：草稿 / 已定稿 / 正式考卷
- 操作：查看答案 / 下载Word / 下载PDF / 复制（跳转组卷，预填参数）

【系统设置】
- 大模型配置：API提供商 / API地址 / API Key / 模型版本 / 温度 / Top P / 最大Token
- 数据字典：科目编码维护（DTM/DTC/DTGV/DTA/通用）
- 系统参数：试题冷却阈值（天数）/ 默认编号前缀 / 难度等级名称自定义
